# Jar's Error Trap

Simple LUA error trap addon for WoW. 

## Installation
Same as any other WoW Addon

### Usage

Click on the minimap icon or type `/jet` to bring up the trap screen:

<img width="1593" height="740" alt="image" src="https://github.com/user-attachments/assets/06d4453a-0f0c-4abe-adf8-d97b251bfd33" />

Also supported is:
`/jet clear`
`/jet test`
